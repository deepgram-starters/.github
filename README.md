# .github

Organization-wide community health files for [deepgram-starters](https://github.com/deepgram-starters).

## What's Here

- **CODE_OF_CONDUCT.md** - Code of conduct for all repositories
- **CONTRIBUTING.md** - Contribution guidelines
- **SECURITY.md** - Security policy and vulnerability reporting
- **CODEOWNERS** - Default code owners (@deepgram-starters/devrel)
- **ISSUE_TEMPLATE/** - Issue templates
- **PULL_REQUEST_TEMPLATE/** - PR templates

These files apply to all repositories in the organization that don't have their own versions.
