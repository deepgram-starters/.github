# Deepgram Starter Apps

Code samples and starter applications demonstrating Deepgram's speech-to-text, text-to-speech, and voice AI capabilities across multiple frameworks and languages.

## üöÄ Popular Starters

**Speech-to-Text**
- [node-transcription](https://github.com/deepgram-starters/node-transcription) - Prerecorded audio transcription with Node.js
- [node-live-transcription](https://github.com/deepgram-starters/node-live-transcription) - Real-time transcription with WebSockets
- [flask-transcription](https://github.com/deepgram-starters/flask-transcription) - Python/Flask transcription starter
- [go-transcription](https://github.com/deepgram-starters/go-transcription) - Go transcription starter

**Text-to-Speech**
- [node-text-to-speech](https://github.com/deepgram-starters/node-text-to-speech) - Convert text to speech with Node.js
- [node-live-text-to-speech](https://github.com/deepgram-starters/node-live-text-to-speech) - Streaming text-to-speech

**Voice Agents**
- [node-voice-agent](https://github.com/deepgram-starters/node-voice-agent) - Build conversational voice AI agents

**Text Intelligence**
- [node-text-intelligence](https://github.com/deepgram-starters/node-text-intelligence) - Sentiment analysis, summarization, topic detection
